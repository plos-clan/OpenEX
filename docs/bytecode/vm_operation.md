# 运算类型指令

包含各种用于运算的指令, 这种指令一般没有操作数, 仅消费操作栈并将结果压回.

## 特殊运算

|      操作码助记符 | 功能                  | 消费     |
|------------:|:--------------------|:-------|
|   `get_ref` | 取出栈顶的两个引用，拼接成一个引用路径 | 两个引用路径 |
| `get_index` | 取出数组指定索引的元素并压入栈顶    | 索引     |

## 算数运算

> 一般将栈顶俩元素取出运算后将结果压回操作栈

| 操作码助记符 | 功能   | 类型要求                      |
|-------:|:-----|:--------------------------|
|  `add` | 加法运算 | `string` `number` `float` |
|  `sub` | 减法   | `number` `float`          |
|  `mul` | 乘法   | `number` `float`          |
|  `div` | 除法   | `number` `float`          |
|  `rmd` | 取余   | `number` `float`          |
